基本函数装饰器(Decorator)🎀

Python装饰器的概念

装饰器是一个特殊的函数，它可以扩展一个普通函数的功能，可以把装饰器中定义的功能辅助于所有被装饰的函数，提高代码的复用率，让代码更简洁。

装饰器的基本定义

装饰器有自己的功能代码，并且传递一个函数作为参数到装饰器中（回调函数）进行调用。

使用装饰器

装饰带参数的函数（传递变量）🔃

装饰带参数的函数，不在被装饰的函数调用时中传递参数，而是在装饰器中的回调函数中传递参数。
